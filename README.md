# Termux Display Manager
Did you always wanted a display manager in termux that is good? Use TDM! it its super cool and fast!
you need a GUI or termux:x11, or vnc, a desktop (xfce4, plasma, mate) download this on your termux and then apt install ./(file)

# Use the installer
You can use the installer if u dont want to make any conflicts with packages not installed, use wget https://raw.githubusercontent.com/Superchavo/termux-display-manager-repo/refs/heads/master/tdminstaller.sh && chmod +x ./tdminstaller.sh && ./tdminstaller.sh

# Why did u make this?
I made this bc i always dreamed of a DM (display manager) for termux, but sadly, lightdm, sddm, gdm, they cant be compilled into termux bc of systemd, but TDM is made in python,

# For automated installers like making termux better or others like installing desktops
instead of the option installer, use the auto installer that downloads the latest version of TDM, use: wget https://raw.githubusercontent.com/Superchavo/termux-display-manager-repo/refs/heads/master/tdmautoinstall.sh && chmod +x ./tdmautoinstall && ./tdmautoinstall
